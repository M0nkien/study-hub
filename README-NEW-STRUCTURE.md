# README NEW STRUCTURE – Fyzika patch

## Doplnenie Fyziky

Tento patch rozširuje predmet Fyzika o nové študijné časti podľa prednášok a prípravy na skúšku.

### Súbory

- PDF prezentácie sú v `files/fyzika/prezentacie/`.
- Obrázky k náčrtom sú v `images/fyzika/`.
- Náhodný test je v `script/fyzika-random-exam.js`.
- Vizuálne doplnky sú v `style/fix.css`.

### Nový princíp stránky

Fyzika už nie je iba zoznam vzorcov. Obsah je rozdelený na:

1. prehľad a okruhy,
2. podrobné odvodenia,
3. riešené príklady,
4. príklady na precvičenie,
5. obrázky/náčrty,
6. teória z prednášok,
7. vzorce,
8. test na 60 minút,
9. checklist.

### Poznámka

Hlavička referátu nie je súčasťou patchu, pretože používateľ ju výslovne nechcel.
