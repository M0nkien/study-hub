OOP

Triedy a objekty

Trieda je predpis. Objekt je konkrétna inštancia. Atribúty opisujú stav, metódy opisujú správanie.

public class Student {
    private String meno;
    private int rocnik;

    public Student(String meno, int rocnik) {
        this.meno = meno;
        this.rocnik = rocnik;
    }
}
