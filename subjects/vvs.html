ESP32 a MicroPython

VVS – teória, príklady, príkazy a obhajoba

ESP32

Základná teória

ESP32 je mikrokontrolér. Programuje sa v MicroPythone a vie pracovať s GPIO, PWM, ADC, UART, I2C, časovačmi a web serverom.

GPIO

Digitálny vstup a výstup

from machine import Pin

led = Pin(21, Pin.OUT)
led.value(1)

button = Pin(2, Pin.IN, Pin.PULL_UP)
PWM

PWM LED a buzzer

from machine import Pin, PWM

pwm = PWM(Pin(5))
pwm.freq(1000)
pwm.duty(512)
ADC

Analógové meranie

from machine import ADC, Pin

adc = ADC(Pin(3))
value = adc.read()
UART

Sériová komunikácia

from machine import UART

uart = UART(1, baudrate=115200, tx=23, rx=15)
uart.write("Ahoj;")
Timer

Časovače

from machine import Timer

def tik(timer):
    print("timer")

tim = Timer(0)
tim.init(period=1000, mode=Timer.PERIODIC, callback=tik)
Web

Web ovládanie robota

ESP32 môže bežať ako jednoduchý web server. Telefón alebo PC otvorí stránku a cez tlačidlá posiela príkazy.

ESP32 piny

Pri každom zapojení si zapíš pin, účel a či je vstup alebo výstup.

LED: Pin(21), Pin(11), Pin(10)
Buzzer: Pin(5)
UART: tx=Pin(23), rx=Pin(15)

MicroPython základy

led = Pin(21, Pin.OUT)
led.value(1)
