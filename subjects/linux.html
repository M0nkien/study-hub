Podrobná teória

Linux Essentials – teória a skúškové poznámky

OS

Operačný systém

Operačný systém sprostredkuje komunikáciu medzi hardvérom, aplikáciami a používateľom. Linux je jadro systému, distribúcia je kompletný systém okolo jadra.

  • kernel = jadro
  • shell = rozhranie pre príkazy
  • distribúcia = kernel + nástroje + balíčky
CLI

Shell a terminál

Shell prijíma príkazy a spúšťa programy. Najčastejší shell je Bash.

pwd        aktuálny adresár
ls -la     podrobný výpis
cd ..      o adresár vyššie
history    história príkazov
Súbory

Súborový systém

Linux má stromovú štruktúru od koreňa /. Dôležité adresáre majú presný účel.

/home   domovské priečinky
/etc    konfiguračné súbory
/bin    základné programy
/var    logy a meniace sa dáta
/tmp    dočasné súbory
Práva

Oprávnenia rwx

Každý súbor má vlastníka, skupinu a práva pre owner/group/others.

r = 4 čítanie
w = 2 zápis
x = 1 spustenie

chmod 755 file
owner rwx, group r-x, others r-x
Text

Pipe a presmerovanie

Pipe spája príkazy. Presmerovanie ukladá výstup do súboru alebo číta vstup zo súboru.

cat file.txt | grep "test"
sort file.txt | uniq
echo "text" > subor.txt
echo "dalsi text" >> subor.txt
Procesy

Procesy a systém

Proces je spustený program. V Linuxe vieš procesy sledovať, ukončovať a filtrovať.

ps aux
top
kill PID
whoami
id

Linux tréning

Čo spraví tento príkaz?

Príkazy podľa kategórií

Pri učení si rozdeľ príkazy podľa účelu: navigácia, súbory, text, práva, procesy a kompresia.

pwd        # kde som
ls -la     # podrobný výpis
cd ..      # o adresár vyššie
mkdir test # vytvor adresár

Príklady s výstupom

Pri skúške sa môže objaviť otázka, čo sa stane po spustení príkazu.

grep "^a" names.txt
# vypíše riadky, ktoré začínajú písmenom a

chmod 755 script.sh
# vlastník rwx, skupina r-x, ostatní r-x

Práva a chmod tréning

Čísla sa skladajú z r=4, w=2, x=1.

7 = 4+2+1 = rwx
6 = 4+2   = rw-
5 = 4+1   = r-x
chmod 640 file.txt = rw-r-----

Pipes a redirect

Pipe posiela výstup jedného príkazu do ďalšieho. Redirect ukladá výstup do súboru.

cat /etc/services | grep "^http" | sort > http-services.txt
