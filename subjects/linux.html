Štúdium

Vysvetlenie príkazov a pojmov

Príkazy sú rozdelené podľa tém a pri každom je krátke vysvetlenie.

Navigácia

Práca s adresármi

pwd

Zobrazí aktuálny pracovný adresár.

ls

Vypíše obsah adresára.

ls -la

Podrobný výpis vrátane skrytých súborov.

cd ..

Presun o adresár vyššie.

~

Domovský adresár používateľa.

Súbory

Práca so súbormi

touch

Vytvorí prázdny súbor alebo zmení čas úpravy.

mkdir

Vytvorí adresár.

cp

Kopíruje súbor/adresár.

mv

Presúva alebo premenuje.

rm -i

Maže súbor s potvrdením.

Text

Filtrovanie textu

cat

Vypíše celý súbor.

less

Prehliadanie dlhého súboru.

grep

Hľadá text alebo regulárny výraz.

sort

Zoradí riadky.

uniq

Odstráni duplicitné susedné riadky.

Práva

Používatelia a oprávnenia

whoami

Aktuálny používateľ.

id

UID, GID a skupiny.

chmod 755

Nastaví práva v oktálovom zápise.

chown

Zmení vlastníka/skupinu.

sticky bit

Chráni cudzie súbory v spoločnom adresári.

Systém

Procesy a kompresia

ps -A

Všetky procesy.

ps aux

Podrobné procesy.

gzip

Kompresia súboru.

zip

ZIP archív.

tar -czf

Vytvorí tar.gz archív.

Pomoc

Dokumentácia

man

Manuálová stránka.

man -k

Vyhľadávanie v manuáloch.

apropos

Hľadanie podľa popisu.

whatis

Krátky popis príkazu.

info

Info dokumentácia.
