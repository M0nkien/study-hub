# Ako zapojiť vylepšený kvíz do Linux stránky

V Linux stránke už máš `quiz-linux.js`, ktorý obsahuje otázky.

Najjednoduchší postup:
1. Do stránky nechaj `quiz-linux.js`.
2. Pridaj nový súbor `quiz-enhanced-template.js`.
3. V HTML sekcii kvízu použi ovládacie prvky podľa `linux-quiz-section-example.html`.

Tento engine vie:
- cvičenie,
- skúšku bez ukázania odpovede počas testu,
- test iba z nesprávnych otázok,
- výsledkovú obrazovku,
- ukladanie chybných otázok do localStorage.

Poznámka:
Ak chceš, môžem ti v ďalšej správe rovno prerobiť celé tvoje konkrétne `linux.html` a `quiz-linux.js` na túto novú verziu.
